- Horizons website for student recruitment
- Horizons is a student recruitment platform that allows employers to browse student projects, discover new talent, and save work that stands out. Students can upload and manage their own work, while admins maintain and moderate the project showcase.

- Luke's role
- The employer role when logged in is on the employer dashboard can view the projects with a drill down option, save a project to another saved projects list and can update it with personal notes. The employer can also delete the project from their saved list.

- Adam's role
- The admin when logged in can update the showcase, delete projects from the showcase, delete the entire showcase and can add projects to the showcase.

- Maria's role
- The user role can create a project and add it to the showcase, they can update the project and edit the details in the project, delete a project frrom the showcase, and can view the showcase.

- Technologies used
- html/css
- javaScript
- php/xampp
- Database-mySql
- additional - docker desktop/github

- Flow of project
- When you go to the login page you choose if you are an employer or an admin and then log in with the correct credentials to get to the 3 different screens for the site.
